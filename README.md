# NetPractice

## Description
NetPractice is a networking project from the 42 curriculum.

The objective of this project is to understand the fundamentals of computer networking by solving network configuration exercises. During the project, I learned how IP addressing, subnet masks, default gateways, routers, and switches work together to enable communication between hosts.

The project consists of 10 levels, each requiring a different network configuration to make the simulated network function correctly.

## Instructions

`./run.sh`

to run the training interface.

If run.sh does not work, start a local web server manually:

`python3 -m http.server 49242`

Then open `http://localhost:49242`

During training, after completing a level, click **Get my config** to get the configuration file.

Place the 10 exported configuration files (one for each level) in the root of the Git repository together with this README.md for submission.
